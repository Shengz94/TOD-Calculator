Calculator

Instructions

This Calculator can do the basic operations: Sum, Subtract, Multiply and Divide.

Other functions are clear values, backspace and negate current number.

There are some control checks while using this calculator:

  • The maximum number of digits (included the comma ".") is 15, exceding it will round the number (if it has decimals) or show an error.
  • You can only type one comma ".".
  • You cannot negate the number "0".

Keyboard shortcut support enabled (you can see the shortcut hovering on each button)
